# Portal Combo - BloxFruits

## How to Use

Ensure the following equipment and mastery levels:

- **Gravity Blade**: 250+ Mastery  
- **Godhuman**: 350+ Mastery  
- **Portal**: 300+ Mastery  
- **Ghoul V3 or v4**
- **Use points in:** Sword
  
---

## Script - Portal Combo

```lua

If this is true, the aimlock used in some combos will only target players.
```
